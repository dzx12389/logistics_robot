## 1. 准备工作

在开始之前，请确保您已经以下条件：

- [ ] 已安装 Python 3.6 或更高版本
- [ ] 已注册快递鸟开发者账号
- [ ] 已获取快递鸟 API 凭证（EBusinessID 和 APIKey）
- [ ] 已下载项目文件

## 2. 文件下载与解压

### 步骤 2.1：下载项目文件
1. 从提供的链接下载项目压缩包
2. 将文件保存到您的电脑上（例如：`C:\Users\Bigboom\Desktop\`）

### 步骤 2.2：解压文件
1. 找到下载的压缩包
2. 右键点击压缩包，选择"解压到当前文件夹"
3. 解压后会得到一个名为名为 `logistics_bot` 的文件夹

### 步骤 2.3：查看文件结构
打开解压后的文件夹，您应该看到以下文件：

```
logistics_bot/
├── logistics_bot_enhanced.py    # 增强版主程序（推荐使用）
├── logistics_bot_standalone.py   # 独立版程序
├── logistics_bot.py             # 原始版程序
├── config.py                    # 配置文件（用于原始版）
├── requirements.txt             # 依赖库列表
├── README.md                    # 项目说明
├── MODIFICATION_GUIDE.md        # 代码修改指南
└── STEP_BY_STEP_GUIDE.md        # 本操作指南
```

## 3. 配置快递鸟凭证

### 步骤 3.1：打开增强版程序文件
1. 找到 `logistics_bot_enhanced.py` 文件
2. 右键点击文件，选择"编辑"或使用 VS Code 打开

### 步骤 3.2：修改配置信息
在文件中找到以下部分：

```python
def __init__(self):
    """
    初始化物流查询机器人
    """
    # TODO: 请替换为您自己的快递鸟凭证
    self.e_business_id = "你的商户ID"  # ← 替换为您的商户ID
    self.api_key = "你的APIKey"        # ← 替换为你的APIKey
```

### 步骤 3.3：保存文件
1. 替换完成后，按 `Ctrl + S` 保存文件
2. 关闭编辑器

## 4. 安装依赖库

### 步骤 4.1：打开命令提示符
1. 按下 `Win + R` 键
2. 输入 `cmd`，然后按回车键

### 步骤 4.2：进入项目目录
输入以下命令，然后按回车键：

```cmd
cd C:\Users\Bigboom\Desktop\logistics_bot
```

### 步骤 4.3：安装依赖
输入以下命令，然后按回车键：

```cmd
pip install -r requirements.txt
```

**如果出现 "pip 不是内部或外部命令" 错误，请尝试：**

```cmd
python -m pip install -r requirements.txt
```

## 7. 常见问题解决

### 问题 7.1：程序启动失败
**错误信息：** `请先配置你的快递鸟凭证！`

**解决方法：**
1. 确保已正确替换 `self.e_business_id` 和 `self.api_key`
2. 检查引号是否正确（必须是英文引号）
3. 确保没有多余的空格

### 问题 7.2：无法识别快递公司
**错误信息：** `无法识别快递公司，请手动输入`

**解决方法：**
1. 输入 `help` 查看支持的快递公司编码
2. 按照提示手动输入快递公司编码
3. 例如：如果是极兔速递，输入 `JTO`

### 问题 7.3：查询失败
**错误信息：** `查询失败：...`

**解决方法：**
1. 检查网络连接是否正常
2. 验证快递单号是否正确
3. 确认快递鸟API凭证是否有效
4. 检查API调用次数是否超过限制

### 问题 7.4：依赖库安装失败
**错误信息：** `pip install 失败`

**解决方法：**
1. 确保Python已正确安装
2. 尝试使用管理员权限运行命令提示符
3. 更新pip：`python -m pip install --upgrade pip`
4. 手动安装：`pip install requests`
