# 📦 快递查询机器人

一个交互式的物流查询机器人，可以在终端中查询真实的物流信息。

## ✨ 功能特点

- 🚀 **实时查询**：查询真实的物流状态和轨迹
- 🤖 **智能识别**：自动识别快递公司，无需手动选择
- 💬 **交互式界面**：终端友好的问答式交互
- 📜 **查询历史**：记录并查看历史查询记录
- 🎨 **美观输出**：使用emoji和格式化输出，提升用户体验
- 🔧 **错误处理**：完善的错误处理和提示信息

## 📋 安装说明

### 1. 克隆项目
```bash
git clone https://github.com/yourusername/logistics-bot.git
cd logistics-bot
```

### 2. 安装依赖
```bash
pip install requests
```

### 3. 配置快递鸟凭证

#### 3.1 获取快递鸟账号
1. 访问 [快递鸟官网](https://www.kdniao.com/)
2. 注册账号并登录
3. 进入 **开发者中心** 获取 `EBusinessID` 和 `APIKey`

#### 3.2 配置凭证
编辑 `logistics_bot.py` 文件，替换以下内容：
```python
# TODO: 请替换为你的快递鸟凭证
E_BUSINESS_ID = "你的商户ID"
API_KEY = "你的APIKey"
```

## 🚀 使用方法

### 1. 运行程序
```bash

```

### 2. 基本操作
- **查询物流**：直接输入快递单号
- **查看历史**：输入 `history`
- **退出程序**：输入 `exit`

## 🔧 常见问题

### 1. 签名错误
- 检查 `EBusinessID` 和 `APIKey` 是否正确
- 确保网络连接正常

### 2. 无法识别快递公司
- 检查快递单号是否正确
- 部分小众快递公司可能无法识别
- 可以手动指定快递公司编码（需要修改代码）

### 3. 查询失败
- 检查网络连接
- 确认快递单号正确
- 检查快递鸟账号是否有余额或权限

## 📈 功能扩展

### 1. 添加语音提示
```bash
pip install pyttsx3
```

在代码中添加：
```python
import pyttsx3

def speak(text):
    engine = pyttsx3.init()
    engine.say(text)
    engine.runAndWait()
```

### 2. 添加数据库存储
可以使用 SQLite 或其他数据库存储查询历史。

### 3. 添加GUI界面
可以使用 Tkinter、PyQt 等框架添加图形界面。
